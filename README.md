# AI Text Summarization Agent

Streamlit web application for text summarization using encoder-decoder models with and without attention mechanisms.

## Features

- Interactive web interface
- Two model comparison (with/without attention)
- Automatic model download from Google Drive
- Real-time processing with progress indicators

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
streamlit run streamlit_app.py
```

## Model Files

Models are automatically downloaded from Google Drive on first run. No manual download needed.

## Requirements

- Python 3.8+
- See `requirements.txt` for package versions
